Your Setup Guide

Everything you need to interact with the IMC platform on Polygon Amoy Testnet.

Step 1 — Install a Wallet
MetaMask Desktop
MetaMask Mobile
MetaMask Extension
Coinbase Wallet
1

Download MetaMask

Go to metamask.io and click Download for Chrome (or Firefox). This installs as a browser extension.

2

Create a New Wallet

Click the MetaMask fox icon in your browser toolbar → Create a new wallet. Set a password and write down your Secret Recovery Phrase. Store it somewhere safe.

⚠ TEST WALLET ONLY: This is for the IMC demo. No real money involved. Never use a real wallet with real funds on testnet.
3

Your Wallet Address

After setup, click the account name to copy your wallet address — it looks like 0xABCD...1234. You'll need this to receive test funds.

1

Download MetaMask App

Search MetaMask in the App Store (iOS) or Google Play (Android). Install the official app by ConsenSys.

2

Create Wallet

Open the app → Create a new wallet → set a password → write down your Secret Recovery Phrase. Confirm it when prompted.

3

Use the In-App Browser

In MetaMask mobile, tap the Browser tab (compass icon). Navigate to innovativeblockchainsolutions.live/imc/ to access the platform. The browser connects directly to your wallet.

4

Copy Your Address

Tap your account name at the top → Copy address. You'll need this for test funds.

1

You Already Have MetaMask

If MetaMask is already installed as a browser extension, click the fox icon in your toolbar to open it.

2

Create a Separate Test Account

Click the account icon → Add account or hardware wallet → Add a new account. Name it IMC Test. This keeps your real accounts separate.

3

Copy the Test Address

Select your new IMC Test account and copy the address shown at the top. Send this to the IMC admin to receive test funds.

1

Install Coinbase Wallet

Go to coinbase.com/wallet and install the browser extension or mobile app. This is separate from the Coinbase exchange app.

2

Create a Self-Custody Wallet

Open Coinbase Wallet → Create a new wallet. Save your recovery phrase. This gives you a standard EVM wallet address.

3

Get Your Address

Tap Receive → Ethereum to see your wallet address. Copy it — it works on all EVM chains including Polygon Amoy.

NOTE: Coinbase Wallet supports Polygon Amoy. You may need to manually add the network in Settings → Networks.

Step 3 — Get Free Test POL

POL is used for gas fees — think of it as the fuel for your transactions. It's free on testnet.

1

Go to the Faucet

Open faucet.polygon.technology in your browser.

2

Select Polygon Amoy

In the network dropdown, select Polygon Amoy. Paste your wallet address and click Submit.

3

Wait for POL

You'll receive ~0.5 POL within 1-2 minutes. This is enough for dozens of transactions.

BACKUP FAUCET: If polygon.technology is slow, try alchemy.com/faucets/polygon-amoy
